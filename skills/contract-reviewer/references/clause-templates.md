# 修改建议模板库

提供可直接使用或稍作填空即可使用的条款模板。模板中 `____` 标记处需合同双方根据具体情况填写。

## 一、检验条款模板

### 推荐模板（区分表面瑕疵与隐蔽瑕疵）

```
1. 检验期间
1.1 表面瑕疵检验：买方应在收到货物后____日内对货物的数量、型号、规格、外观等表面瑕疵进行检验。如有异议，应在该期限内书面通知卖方，逾期视为货物表面瑕疵符合约定。

1.2 隐蔽瑕疵检验：买方应在收到货物后____日内对货物的内在质量等隐蔽瑕疵进行检验。如有异议，应在该期限内书面通知卖方，逾期视为货物质量符合约定。隐蔽瑕疵的检验期间不得少于____日。

2. 检验方式
2.1 表面瑕疵检验由买卖双方在交货地点共同进行，以现场清点、目测为主要方式。检验完成后双方应签署《货物交接验收单》作为货物交付的凭证。

2.2 隐蔽瑕疵检验由□双方共同委托检验机构 / □买方单方委托检验机构进行检验。若由买方单方委托检验，卖方对检验结果有异议的，可于收到检验报告后____日内申请复检，复检机构由双方协商确定。

2.3 检验费用由送检方先行垫付。检验结果符合约定的，检验费用由□买方 / □卖方承担。检验结果不符合约定的，检验费用由卖方承担。

3. 异议处理
3.1 买方在约定的检验期间内提出异议的，卖方应在收到异议通知后____日内予以答复并提供解决方案。逾期未答复的，视为认可买方提出的异议。

3.2 因货物质量问题发生争议，经检验确认货物质量不符合约定的，卖方应负责更换符合约定质量标准的货物，因此而产生的全部费用（包括但不限于检验费、运输费、仓储费等）由卖方承担。
```

## 二、交付条款模板

### 推荐模板（含运输）

```
1. 交付时间：卖方应于____年____月____日前将本合同约定的全部货物交付给买方。卖方可分批次交付的，具体交付时间及数量如下：
   第一批次：____年____月____日前，交付____（数量）；
   第二批次：____年____月____日前，交付____（数量）。

2. 交付地点：____（具体地址）。

3. 交付内容：卖方交付货物时应当同时交付以下文件和资料：
   （1）产品合格证；
   （2）质量保证书；
   （3）产品使用说明书；
   （4）装箱单；
   （5）增值税专用发票；
   （6）其他：____。

4. 运输与装卸
4.1 货物由卖方负责运输至交付地点，运输费用由□卖方 / □买方承担。
4.2 货物在运输途中的毁损、灭失风险，在货物交付给买方之前由卖方承担，交付之后由买方承担。
4.3 装货义务由卖方承担，卸货义务由□卖方 / □买方承担。
```

## 三、价款与支付条款模板

### 推荐模板（分期支付）

```
1. 价款
1.1 本合同项下货物总价款为人民币____元（大写：____元整），含增值税。
1.2 上述价款为固定总价，除本合同另有约定外，合同履行期间不因任何因素调整。

2. 支付方式
2.1 买方采用银行转账方式将合同款项支付至卖方以下银行账户：
   开户行：____
   账户名称：____
   账号：____

2.2 若卖方变更收款账户，应提前____个工作日书面通知买方，否则买方按原账户付款视为已履行付款义务。

3. 付款安排
3.1 预付款：本合同生效后____个工作日内，买方支付合同总价款的____%，即人民币____元。
3.2 到货款：卖方将全部货物交付至本合同约定的交付地点并经买方签收后____个工作日内，买方支付合同总价款的____%，即人民币____元。
3.3 验收款：货物通过最终验收合格后____个工作日内，买方支付合同总价款的____%，即人民币____元。
3.4 质保金尾款：质量保证期届满且无质量问题或卖方已履行全部质保义务后____个工作日内，买方支付合同总价款的____%，即人民币____元。

4. 发票
4.1 卖方在收到每笔款项后____个工作日内向买方开具等额增值税专用发票。
4.2 卖方未按约定开具发票的，买方有权顺延支付下一笔款项，且不承担逾期付款的违约责任。
```

## 四、违约责任条款模板

### 卖方违约责任

```
1. 卖方逾期交货的，每逾期一日，应按逾期交付货物对应价款的____‰向买方支付违约金。逾期超过____日的，买方有权单方解除本合同，卖方除返还已收取的全部货款外，还应按合同总价款的____%向买方支付违约金。

2. 卖方交付的货物不符合本合同约定的质量标准且无法通过更换达到约定标准的，买方有权要求卖方退还该部分货物的价款，卖方应按该部分货物价款的____%向买方支付违约金。

3. 卖方交付的货物数量不足的，卖方应在收到买方通知后____日内补足，逾期未补足的，按逾期交货处理。

4. 卖方未按照本合同约定提供售后服务（如安装、调试、培训、保修等）的，买方有权自行或委托第三方提供相关服务，因此产生的合理费用由卖方承担。买方有权从未支付的货款或质保金中直接扣除相关费用。
```

### 买方违约责任

```
1. 买方逾期付款的，每逾期一日，应按逾期支付金额的____‰向卖方支付违约金。逾期超过____日的，卖方有权单方解除本合同，并要求买方按合同总价款的____%支付违约金。

2. 买方无正当理由拒绝接收货物的，应承担卖方因此产生的仓储费、运输费等实际损失，并按合同总价款的____%向卖方支付违约金。

3. 买方未按约定时间进行验收的，自约定验收期限届满之日起，视为货物已经通过验收，买方应按本合同约定支付相应款项。
```

## 五、争议解决条款模板

### 诉讼版本

```
因本合同引起的或与本合同有关的任何争议，双方应友好协商解决。协商不成的，任何一方均有权向□合同签订地 / □合同履行地 / □被告住所地 / □原告住所地 / □标的物所在地（择一填写，不得同时选择）有管辖权的人民法院提起诉讼。
```

### 仲裁版本

```
因本合同引起的或与本合同有关的任何争议，双方应友好协商解决。协商不成的，任何一方均有权提交____仲裁委员会（填写具体完整的仲裁机构名称，如"中国国际经济贸易仲裁委员会"）按照该会届时有效的仲裁规则进行仲裁。仲裁裁决是终局的，对双方均有约束力。
```

> 注意：禁止在合同中同时出现诉讼和仲裁两种约定。

## 六、赔偿上限与免责条款模板

```
1. 赔偿责任限制
1.1 任何一方违反本合同约定给对方造成损失的，应赔偿对方的全部实际损失（包括直接损失和可预见的间接损失），赔偿总额不超过本合同总价款的____倍。
1.2 下列损失不在赔偿责任限制范围内：
    （1）因一方故意或重大过失造成的损失；
    （2）因一方违反保密义务或知识产权条款造成的损失；
    （3）人身伤亡造成的损失。

2. 不可抗力
2.1 因不可抗力导致不能履行合同的，受影响方应在不可抗力事件发生后____日内书面通知对方，并在____日内提供相关证明文件。
2.2 未按照约定期限通知并提供证明的，不免除该期间的违约责任。
```

## 七、知识产权与保密条款模板

```
1. 知识产权归属
1.1 卖方为履行本合同而向买方提供的技术资料、图纸、软件等知识产权仍归卖方所有，买方仅享有为使用所购货物而必要的许可使用权。
1.2 因履行本合同产生的定制开发成果及技术改进，知识产权归□买方 / □双方共有 / □卖方所有。归卖方或双方共有的，买方享有永久的、不可撤销的、免费的全球使用许可。

2. 保密义务
2.1 双方在履行本合同过程中知悉的对方商业秘密、技术信息和合同条款本身，均属保密信息。保密期限自本合同签署之日起至合同终止后____年。
2.2 下列信息不适用保密义务：（1）接收方在收到前已合法持有且有书面证明；（2）非因接收方原因已公开的信息；（3）依法须向政府部门或司法机构披露的信息，但披露前应事先通知提供方。
2.3 违反保密义务的，违约方应向守约方支付不少于人民币____元的违约金，违约金不足以弥补损失的，守约方有权继续追偿。
```

## 八、合同变更与解除模板

```
1. 合同的变更
对本合同的任何修改或补充，均应由双方协商一致后以书面形式作出，并由双方签字盖章后生效。

2. 合同的解除
2.1 经双方协商一致的，可以解除本合同。
2.2 任何一方有以下情形之一的，另一方有权单方解除本合同，并要求违约方承担违约责任：
    （1）卖方逾期交付货物超过____日的；
    （2）买方逾期支付货款超过____日的；
    （3）一方存在其他根本违约行为，致使合同目的无法实现的。

3. 解除后的处理
3.1 合同解除后，尚未履行的终止履行。
3.2 卖方应在收到解除通知后____日内退还买方已支付的全部款项，逾期未退还的，按日万分之____支付逾期利息。
3.3 买方应在收到解除通知后____日内返还已收到的货物，返还费用由违约方承担。买方怠于返还的，承担货物毁损灭失风险和仓储费用。
```
