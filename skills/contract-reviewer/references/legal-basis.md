# 买卖合同审查高频法条速查

## 《民法典》合同编 — 买卖合同（第595条-第647条）

| 法条 | 内容要点 | 审查应用场景 |
|------|---------|------------|
| 第595条 | 买卖合同定义：出卖人转移所有权，买受人支付价款 | 确定合同是否构成买卖关系 |
| 第596条 | 买卖合同一般内容（标的物名称、数量、质量、价款、履行期限/地点/方式、包装方式、检验标准和方法、结算方式、文字效力） | 合同结构完整性检查 |
| 第599条 | 出卖人交付提取标的物单证以外的有关单证和资料 | 交付内容审查——附随资料是否列明 |
| 第601条 | 约定交付期限的，出卖人可在期限内任何时间交付 | 交付时间条款审查 |
| 第603条 | 交付地点确定规则（需运输→第一承运人；不需要→已知地点或出卖人营业地） | 交付地点缺失时的默认规则 |
| 第604条 | 风险转移：交付前由出卖人承担，交付后由买受人承担 | 风险转移条款审查 |
| 第619条 | 包装方式：按约定→通用方式→足以保护标的物的方式 | 包装条款审查 |
| 第620条 | 买受人应在约定期限内检验，无约定的应及时检验 | 检验条款审查 |
| 第621条 | 买受人怠于通知 → 视为数量或质量符合约定；未约定期限的→合理期限或2年或质保期 | 检验期间条款审查（核心） |
| 第646条 | 其他有偿合同无规定的，参照适用买卖合同有关规定 | 类买卖合同审查依据 |

## 买卖合同司法解释（法释〔2020〕17号）

| 条款 | 内容要点 | 审查应用场景 |
|------|---------|------------|
| 第7条 | "有关单证和资料"范围：保险单、保修单、发票、合格证、质保书、鉴定书、检疫书、产地证明、说明书、装箱单等 | 交付内容条款完整性检查 |
| 第8条 | 增值税专用发票不能单独证明交付事实；普通发票依约定或习惯可作为付款凭证 | 发票与付款/交付的关系 |
| 第17条 | "合理期间"认定因素：交易性质/目的/方式/习惯、标的物种类/数量/性质、安装使用情况、瑕疵性质、检验方法/难度等 | 检验期间条款合理性判断 |
| 第18条 | 检验期间过短→仅认定为外观瑕疵异议期间；短于法定期间→以法定期间为准 | 检验期间过短的风险判断 |
