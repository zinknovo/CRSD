# 买卖合同审查规则（货物买卖）

基于《民法典》合同编及买卖合同司法解释，结合实务经验整理。

## 速查清单（逐项不跳过）

| # | 维度 | 必查子项 |
|---|------|---------|
| 1 | 主体资质 | 法人信息一致、实际经营地、分公司授权、履约能力排查 |
| 2 | 标的物 | 特定化、标准计量单位、多品类须附件清单 |
| 3 | 价款支付 | 单价总价含税币种、付款节奏、发票时间类型 |
| 4 | 交付 | 时间/地点/方式、运费承担、附随资料、装卸义务 |
| 5 | 验收异议 | 表面/隐蔽区分、检验期间合理性、检验方式机构费用 |
| 6 | 风险转移 | 转移节点、运输途中风险、包装适配性 |
| 7 | 违约责任 | 覆盖双方全部义务、违约金类型匹配、解除权约定 |
| 8 | 赔偿免责 | 赔偿上限、免责范围（排除故意/重大过失→无效） |
| 9 | 知识产权 | 定制成果归属、保密范围期限 |
| 10 | 争议解决 | 唯一性（诉讼/仲裁择一）、管辖明确 |
| 11 | 变更解除 | 书面形式、解除条件、善后安排 |
| 12 | 生效其他 | 生效条件、附件列明、通知送达 |

---

## 一、合同主体审查要点

1. **签约资质特殊情形**：分公司签约的，必须核实总公司营业执照及授权文件；自然人以个体工商户名义签约的，核实营业执照。
2. **履约能力排查路径**：除裁判文书网、失信名单外，还需审查对方是否被列入经营异常名录（国家企业信用信息公示系统）。发现涉诉多或已列入失信名单的，提示委托人慎重考虑或采取防范措施（如提高预付款比例、要求提供担保、缩短付款周期）。

## 二、标的条款审查要点

1. **核心原则**：标的物必须特定化——合同描述能唯一确定买卖的货物。
2. **必要要素**：品名 + 规格 + 型号 + 等级 + 产地 + 数量（视货物类型增删）。
3. **数量单位**：禁止使用"包、箱、袋、捆、车"等非标准计量单位，必须使用标准单位（如 KG、吨、件等）。
4. **多品类货物**：以附件形式列明标的物清单。

## 三、质量条款审查要点

1. **质量标准必须清晰**：
   - 国标/行标：注明标准名称 + 标准代码
   - 企标：将该标准作为附件
   - 约定标准：条款描述需清晰明确
2. **专业性强的货物**：与委托人沟通后确定合理质量标准。
3. **卖方注意**：不宜约定高于货物客观质量的标准，否则自设违约陷阱。

## 四、包装方式条款审查要点

1. 约定明确的包装方式。
2. 包装费用承担方。
3. 依据《民法典》第619条，无约定时按通用方式包装，无通用方式的采取足以保护标的物的包装方式。
4. **包装适配性**：检查约定的包装方式是否足以保护特定货物。电子设备、精密仪器、易碎品需要防震/防潮/防静电包装；"木托简易包装"对换电柜、服务器等设备可能不足。包装不足以保护货物的，标的"一般"——运输途中的损坏风险实质上未被覆盖。

## 五、交付条款审查要点

### 交付时间
- 具体日期或合理期间。
- 分期交付：明确交付次数、每次时间及数量。
- 买方需要准备时间的，约定卖方提前告知义务。

### 交付地点
- 必须约定明确地址（《民法典》第603条）。
- 需运输的货物：明确运费承担方。

### 交付内容
- 货物本体 + 附随资料（保险单、保修单、普通发票、增值税专用发票、产品合格证、质量保证书、质量鉴定书、品质检验证书、产品进出口检疫书、原产地证明书、使用说明书、装箱单等）。
- 需要装卸的货物：明确装卸义务方。注意"支持"/"协助"类表述——"需方支持下货"不等于"需方负责卸货"，须区分"负责"和"协助"。表述模糊的，标为"一般"。

## 六、检验条款审查要点（高发争议区域）

### 检验期间
1. **必须约定明确的检验期间**——无约定的，"合理期间"认定标准复杂，容易产生争议。
2. **区分表面瑕疵与隐蔽瑕疵**：
   - 表面瑕疵（外观、型号、规格、花色）：约定较短检验期（即时/数日）。
   - 隐蔽瑕疵（内在质量）：约定较长的合理检验期。
3. **检验期间不宜过短**（《买卖合同司法解释》第18条）：若约定期间过短无法完成检验，法院仅认定为外观瑕疵异议期间。

### 检验方式
1. 数量和表面瑕疵：现场清点、双方共同检验。
2. 隐蔽瑕疵：明确检验机构、送检程序、结果确认、费用承担。
3. 双方共同委托检验：提前约定机构和配合义务。
4. 单方检验：约定对方不认可时的解决途径（通常为复检）。
5. 检验费用：一般送检方垫付，结果符合约定由买受人承担，不符合由卖方承担。

### 法律后果
- 买方怠于通知 → 视为标的物数量或质量符合约定（《民法典》第621条）。
- 未约定检验期间 → 适用"合理期间"或最长2年或质保期（《民法典》第621条第2款）。

## 七、售后服务条款审查要点

1. 适用情形：需要长期使用或技术性较强的货物。
   - **注意定级**：将放弃质保作为对买方的惩罚条款（如"买方逾期验收的，视为放弃质保服务"），虽然不合理，但质保作为法定义务不会因合同约定而轻易被免除。此类条款标的"一般"——建议修改，但不构成严重法律风险。
2. 审查内容：
   - 售后服务方式是否明确（安装、调试、培训、保修、升级）。
   - 各项服务的可操作性（时限、次数、范围、师资、场地等逐一明确）。
   - **买方自行安装的缺口检查**：若合同约定由买方（需方）自行安装但货物技术性较强，检查是否约定了卖方提供安装指导、技术培训、安装手册等配套义务。无配套指导义务的自行安装条款，标的"一般"——安装不当的风险完全由买方承担，且可能影响质保的有效性。

## 八、价款及支付条款审查要点

1. **计价标准**：单价、总价、是否含税、币种，缺一不可。
2. **外币支付**：约定汇率变动风险承担。
3. **支付方式**：建议转账（保留支付痕迹），约定收款方完整银行账户信息。
4. **付款节奏**：
   - 推荐方式——分期支付（合同签订→预付款；货到→部分款；验收合格→部分款；售后服务结束→尾款）。
   - 其他方式——一次性支付、先款后货、先货后款，取决于双方谈判地位。
5. **发票**：
   - 明确开票时间（付款前/后）。
   - 普通发票可作为付款凭证（《买卖合同司法解释》第8条）。
   - 专用发票不能单独作为交付货物的证明。

## 九、违约责任条款审查要点

### 全面性要求
- 每项合同义务都必须有对应的违约责任，禁止使用空泛概括性条款。

### 卖方违约责任（至少应覆盖）
1. 逾期交付：每逾期一天，按总货款 `____%` 支付违约金；逾期超过 `____` 天，买方有权解除合同。
2. 质量不符：更换、折价 + 适当违约金；重要质量瑕疵可约定拒收/解除合同 + 赔偿损失。
3. 未提供售后服务：买方有权向第三方购买同类服务，费用向卖方追索或从质保金中扣除。

### 买方违约责任（至少应覆盖）
1. 怠于收货：每逾期一天，按总货款 `____%` 支付违约金 + 承担仓储费/运输费。
2. 怠于验收：视为货物符合约定。
3. 逾期付款：每逾期一天，按拖欠货款的 `____%` 支付违约金；逾期超过 `____` 天，卖方有权解除合同。

### 合理性要求
1. 违约责任类型与违约行为匹配（表面瑕疵→补足更换、逾期→累计违约金）。
2. 违约金比例合理，尊重谈判地位但不放弃约定。

## 十、赔偿责任上限与免责条款审查要点

1. **赔偿责任上限**：
   - 《民法典》第584条（可预见规则）：违约赔偿额不超过订立合同时可预见的损失。
   - 约定"赔偿总额不超过合同价款"对违约方有利——若己方为守约方，争取删除或提高上限（如约定为合同价款的2-3倍）。
   - 违约金远低于实际损失时，可依《民法典》第585条申请增加。
2. **免责范围审查**：
   - 免责条款不得排除故意或重大过失责任（《民法典》第506条）。
   - 不可抗力条款须覆盖通知和减损义务——未及时通知的不免除通知前的损失。
   - 警惕过度免责：如"卖方在任何情况下均不承担间接损失"可能排除主要合同义务责任。

## 十一、知识产权与保密条款审查要点

1. **知识产权归属**（适用于定制开发、技术交付类买卖合同）：
   - 定制开发成果的知识产权须明确归属（买方所有/双方共有/卖方所有）。默认为委托创作时著作权归受托人（《著作权法》第19条）。
   - 卖方交付的技术文件、图纸等知识产权仍归卖方，但须明确买方为使用货物所需的使用许可范围。
2. **保密条款**：
   - 保密信息范围是否明确（技术信息、商业信息、合同条款自身）。
   - 保密期限建议不短于合同期满后2年。
   - 保密义务例外：依法披露、已公开信息、接收方合法持有信息不属违约。
   - 是否约定了保密违约金或赔偿。

## 十二、争议解决条款审查要点

1. **方式唯一性原则**：只能选择诉讼或仲裁之一，同时约定 → 仲裁协议无效（仲裁法司法解释第7条）。
2. **诉讼**：管辖法院必须明确（被告住所地、合同履行地、合同签订地、原告住所地、标的物所在地中与实际争议有实际联系的法院），不得违反级别管辖和专属管辖。
3. **仲裁**：仲裁机构名称必须准确，同一城市有两家以上仲裁机构的必须明确具体哪一家。

## 十三、常见风险模式

### 模式1：检验期间过短
合同约定15天检验期但货物需要30天第三方检测 → 法院仅认定为外观瑕疵异议期间，隐蔽瑕疵隐患仍在。

### 模式2：计量单位非标准
约定以"车"、"箱"为计量单位 → 履行时产生数量争议，无明确标准可依。

### 模式3：争议解决条款无效
同时约定"可向仲裁委员会申请仲裁或向人民法院起诉" → 仲裁协议无效。

### 模式4：违约金缺位
仅约定"违约方应承担违约责任" → 无实质约束力，守约方维权成本高。

### 模式5：付款条件模糊
约定"货到后付款" → 未明确"货到"是指货物发出、抵达还是验收合格，容易产生争议。

### 模式6：质量标准引用不明
约定"符合国家标准" → 未注明标准名称和代码，双方可能指向不同标准。

### 模式7：交货时间依赖单方通知
约定"交货时间以甲方通知为准"或类似表述 → 交货时间不确定，若通知方怠于通知则交货无限延后。建议约定最迟通知期限和最迟交货期限。若同时约定具体日期但该日期已过，标的"一般"——合同履行已客观上逾期。

### 模式8：付款条件冲突
同时存在两个或多个不同的付款触发条件（如"验收合格后____日内"与"货到____日内"并存），或"六个月内"未明确起算时点（到货日？验收合格日？）。付款条件冲突或起算点模糊的，标的"重要"，建议将付款节点统一为单一明确的时点。
